# XmlTreeVisualizer
A very basic XML tree visualizer written in Python, works best with smaller XML files

Usage:
```py
import XmlTreeVisualizer as XML

visualizer = XML.XmlTreeVisualizer("path_to_xml_file")
diagram = visualizer.display_tree()

print(diagram)
```
